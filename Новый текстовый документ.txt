git init инициализируешь папку
git remote add origin https://github.com/micizi/test.git 

git status проверка файлов
git add . добавить файл (или без точки)
git commit -m 'описание для себя к файлу'

git push отправить все файлы из папки которые закомичены в репозиторий
возможно вставляешь нижнюю строку что предложит и после введёшь логин, пароль
