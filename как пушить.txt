git init инициализируешь папку
git remote add origin https://github.com/micizi/test.git это берёшь у чела

git status проверка файлов
git add . добавить файл
git commit -m 'описание для себя к файлу'

git push отправить все файлы из папки которые закомичены в репозиторий
возможно вставляешь нижнюю строку что предложит
git push --set-upstream origin main
и после введёшь логин, пароль
